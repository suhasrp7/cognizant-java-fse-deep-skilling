# Library Management - Exercise 4 (Maven Configuration)

## Description

This exercise demonstrates the creation and configuration of a Maven project for a Library Management Application using the Spring Framework.

The project includes Spring dependencies and Maven compiler configuration.

## Objectives

- Create a Maven project named LibraryManagement
- Add Spring Framework dependencies
- Configure Maven Compiler Plugin
- Set Java version to 1.8

## Technologies Used

- Java
- Maven
- Spring Framework

## Dependencies Added

- Spring Context
- Spring AOP
- Spring WebMVC
