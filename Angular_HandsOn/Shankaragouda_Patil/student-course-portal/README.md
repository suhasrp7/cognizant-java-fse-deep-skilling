## Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd student-course-portal
```

Install dependencies

```bash
npm install
```

Run the application

```bash
ng serve
```

Open:

```
http://localhost:4200
```
