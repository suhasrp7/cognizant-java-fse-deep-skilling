# Angular Hands-On 3 – Structural Directives, Attribute Directives & Custom Directives/Pipes

## Objective
Implement Angular structural directives, attribute directives, custom directives, and custom pipes in the Student Course Portal application.

## Features
- Structural Directives
  - *ngIf
  - *ngFor
  - trackBy
- Attribute Directives
  - ngClass
  - ngStyle
- Custom Directive
  - Highlight Directive
- Custom Pipe
  - Course Title Pipe
- Dynamic Course Card Display
- Parent-Child Component Communication using @Input and @Output
- EventEmitter for Enrollment

## Technologies Used
- Angular 21
- TypeScript
- HTML
- CSS
- Node.js

## Output

The `Output` folder contains screenshots demonstrating:

- Student Course Portal home page
- Course cards with `ngClass` and `ngStyle`
- Custom Directive and Custom Pipe implementation

## Run the Project

```bash
npm install
ng serve
```

Open the application in your browser:

```
http://localhost:4200
```
