# Hands-On 5 - Reactive Forms

## Objective

Develop a Reactive Course Enrollment Form using Angular Reactive Forms with FormBuilder, FormGroup, FormArray, Custom Validators, and Async Validators.

## Features Implemented

- Created Reactive Enrollment Form Component
- Configured Angular Routing
- Used ReactiveFormsModule
- Implemented FormBuilder and FormGroup
- Added Built-in Validators
- Implemented Custom Validator for Course ID (XX101 is not allowed)
- Implemented Custom Validator for Student Name (No Leading Space)
- Implemented Async Validator for Email (test@example.com)
- Implemented Dynamic Skills using FormArray
- Added Add Skill and Remove Skill functionality
- Displayed Validation Messages
- Submitted Form Data to Browser Console

## Technologies Used

- Angular
- TypeScript
- HTML
- CSS
- RxJS

## Output

The application allows users to:

- Enter Student Details
- Validate Student Name using a Custom Validator
- Validate Course ID using a Custom Validator
- Validate Email using an Async Validator
- Add Multiple Skills dynamically
- Remove Skills dynamically
- Submit the Reactive Form
- View Submitted Data in Browser Console
