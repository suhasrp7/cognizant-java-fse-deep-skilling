# Hands-On 5 - Reactive Forms

## Objective

Develop a Reactive Course Enrollment Form using Angular Reactive Forms with FormBuilder, FormGroup, FormArray, and Custom Validators.

## Features Implemented

- Created Reactive Enrollment Form Component
- Configured Angular Routing
- Used ReactiveFormsModule
- Implemented FormBuilder and FormGroup
- Added Built-in Validators
- Implemented Custom Validator
- Implemented Dynamic Skills using FormArray
- Added Add Skill and Remove Skill functionality
- Displayed Validation Messages
- Submitted Form Data to Browser Console

## Technologies Used

- Angular
- TypeScript
- HTML
- CSS

## Output

The application allows users to:

- Enter Student Details
- Add Multiple Skills
- Validate Form Inputs
- Submit the Reactive Form
- View Submitted Data in Browser Console
