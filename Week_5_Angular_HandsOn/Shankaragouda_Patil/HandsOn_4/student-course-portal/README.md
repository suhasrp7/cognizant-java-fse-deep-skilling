# Hands-On 4 - Template-Driven Forms and Validation

## Objective

Develop a Template-Driven Enrollment Form in Angular using FormsModule and implement form validation.

## Features Implemented

- Created Enrollment Form Component
- Configured Angular Routing
- Used Template-Driven Forms
- Implemented Two-Way Data Binding using ngModel
- Added Form Validation
- Displayed Validation Error Messages
- Disabled Submit Button for Invalid Form
- Displayed Success Message after Submission
- Added Reset Button
- Applied Red/Green Border Styling for Invalid/Valid Fields

## Technologies Used

- Angular
- TypeScript
- HTML
- CSS

## Output

The application displays a Course Enrollment Form with validation for:
- Student Name
- Email
- Course ID
- Preferred Semester
- Terms & Conditions

The form successfully submits valid data and displays a success message.
