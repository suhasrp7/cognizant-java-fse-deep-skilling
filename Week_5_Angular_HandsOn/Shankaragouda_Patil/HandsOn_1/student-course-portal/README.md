# Angular Hands-On 1 – Student Course Portal

## Objective
Set up the Angular development environment, create a new Angular project, and build a simple Student Course Portal using standalone components.

## Features
- Angular project setup
- Standalone components
- Header component
- Home component
- Course List component
- Student Profile component
- Basic navigation UI

## Technologies Used
- Angular 21
- TypeScript
- HTML
- CSS
- Node.js

## Output
Output screenshot is available in the `Output` folder.

## Run the Project

```bash
npm install
ng serve
```

Open: `http://localhost:4200`
