# HandsOn 7 - Angular Routing & Route Guards

## Author
**Shankaragouda Patil**

## Description
This project demonstrates Angular Routing, Nested Routes, Route Parameters, Query Parameters, Lazy Loading, and Route Guards (CanActivate & CanDeactivate) in the Student Course Portal application.

## Technologies Used
- Angular 20
- TypeScript
- HTML
- CSS

## Features
- Routing
- Nested Routing
- Route Parameters
- Query Parameters
- Lazy Loading
- CanActivate Guard
- CanDeactivate Guard

## How to Run

```bash
npm install
ng serve
```

Open in browser:

```
http://localhost:4200
```
