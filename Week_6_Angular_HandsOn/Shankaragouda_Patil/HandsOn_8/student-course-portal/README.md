# HandsOn 8 - Angular HttpClient, RxJS and HTTP Interceptor

## Author
**Shankaragouda Patil**

## Description
Implemented Angular HttpClient for REST API communication using JSON Server. Integrated RxJS operators for data processing and error handling, and configured an HTTP Interceptor to log HTTP requests.

## Technologies Used
- Angular 20
- TypeScript
- HTML
- CSS
- RxJS
- JSON Server

## Features
- HttpClient Integration
- GET, POST, PUT, DELETE Operations
- RxJS Operators (map, tap, retry, catchError)
- HTTP Interceptor
- JSON Server Integration

## How to Run

```bash
npm install
json-server --watch db.json --port 3000
ng serve
```

Open:
http://localhost:4200
